# Redis中国用户组 www.redis.cn #

## 项目简介 ##
本项目是将英文版[redis.io](http://redis.io)翻译为[redis.cn](http://redis.cn)的项目，是由中国用户组里面的一线工程师组建而成的[翻译团队](/aboutus.html)。更多技术问题交流欢迎访问[Redis中文论坛](http://bbs.redis.cn)。
